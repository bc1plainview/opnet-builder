# Swap UI Blueprint

Scaffold a React frontend that connects to MotoSwap for OP20-to-OP20 token swaps on Bitcoin Layer 1.

## What You Get

| Component | Description |
|-----------|-------------|
| **React Frontend** | Vite + TypeScript swap interface with OP_WALLET integration |

## Features
- Token selector with balance display
- Swap interface with slippage settings
- Price impact calculation
- Transaction confirmation flow
- OP_WALLET integration via @btc-vision/opwallet
- Responsive dark theme
