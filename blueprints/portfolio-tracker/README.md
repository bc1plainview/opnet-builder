# Portfolio Tracker Blueprint

Scaffold a React frontend that displays a user's OPNet token balances and transaction history on Bitcoin Layer 1.

## What You Get

| Component | Description |
|-----------|-------------|
| **React Frontend** | Vite + TypeScript portfolio dashboard with OP_WALLET integration |

## Features
- Wallet connection via OP_WALLET
- Multi-token balance display (queries OP20 contracts)
- Transaction history view
- USD value estimation (via external price feeds)
- Responsive dark theme
